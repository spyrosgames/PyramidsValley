HyperGlyph Gesture Recognition System v1.0

Unicorn Forest Games 2012

Check out the example scene to get started.
To make new Gesture Sets Click [Create] -> [Gesture Set]
Then configure everything through the inspector.

For Script Reference go to : http://unicornforestgames.com/UnityPlugins/HyperGlyph/ScriptReference
For support email: jacob@UnicornForestGames.com

Note to use with UnityScript you must move to folder called HyperGlyphLib to StandardAssets or Plugins.
